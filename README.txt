TR: Programın Kullanımı:

Programı yalnızca Kick kanalınızdaki "Etkinlik Akışı" sayfasına erişenler kullanabilir.

Programı çalıştırdıktan sonra, programın dinleyeceği Kick kanalının adını eksiksiz olarak girin.

Açılan sitede (tercihen Chrome kullanın; diğer tarayıcılarda çalışma garantisi yoktur) yer işaretleri çubuğuna sağ tıklayın ve "Sayfa ekle..." seçeneğine tıklayın. (Eğer yer işaretleri çubuğu görünmüyorsa tarayıcı ayarlarından açabilirsiniz.)

Açılan pencerede, Ad kısmına istediğiniz ismi yazın. URL kısmına ise "Key trigger.txt" adlı dosyada bulunan kodu kopyalayıp yapıştırın ve kaydedin.

Oluşturduğunuz yer işaretine sayfa yüklendikten sonra bir kez tıklayın. (Birden çok tıklamak programın defalarca çalışmasına neden olur. Birden fazla tetiklemeye neden olur)

Bu yer işaretini ekledikten sonra bir sonraki kullanımlarda sadece sayfayı açıp yüklendikten sonra yer işaretine tıklamanız yeterli olacaktır.

Buradaki işlemi tamamladıktan sonra ana programa geri dönün.

Reward name (Ödül adı) kısmına, Kick’teki sadakat puanı ödülünün tam adını girin.

Key sequence (Tuş dizisi) kısmına, tetiklemek istediğiniz tuş(lar)ı ve her tuş basımı sonrası beklenecek süreyi saniye cinsinden, örneğe uygun olarak yazın. (Örnekteki noktalama işaretlerine dikkat edin.)

Tek bir tuş tetikleniyorsa bile, 0.1 saniye gibi küçük bir gecikme eklemeniz gerekmektedir, çünkü art arda gelen ödüllerde çakışmalar olabilir.

Add Mapping (Eşleştirme Ekle) tuşuna basarak eşleşmeyi kaydedin.

Clear Mappings (Eşleşmeleri Temizle) ile tüm eşleşmeleri silebilirsiniz.

Her şey hazır olduğunda Start (Başlat) butonuyla sunucuyu çalıştırabilirsiniz.

Programı kapatmadan sunucuyu geçici olarak durdurmak isterseniz, Stop (Durdur) butonunu kullanabilirsiniz.

Keyifli kullanımlar!

Not: Programın tarayıcı kısmının çalıştığına emin olmak isterseniz yer işaretine tıkladığınız sayfada ctrl+shift+j tıklayarak console'yi açın. Burada eğer "Chat container found. Messages are being monitored..." yazısını görüyorsanız program sorunsuz çalışmıştır demektir.



ENG: How to Use the Program:

The program can only be used by those who have access to the "Activity Feed" page of your Kick channel.

After running the program, enter the exact name of the Kick channel that the program will monitor.

On the opened site (preferably use Chrome; no guarantee it will work on other browsers), right-click on the bookmarks bar and select "Add page...". (If the bookmarks bar is not visible, you can enable it from your browser settings.)

In the opened window, enter any name you like in the Name field. In the URL field, copy and paste the code from the file named "Key trigger.txt" and save it.

After the page loads, click once on the bookmark you created. (Clicking multiple times will cause the program to run multiple times and trigger multiple activations.)

Once you have added this bookmark, for future use you just need to open the page and click the bookmark after it loads.

After completing this step, return to the main program.

In the Reward name field, enter the exact name of the channel point reward on Kick.

In the Key sequence field, write the key(s) you want to trigger and the delay after each key press in seconds, following the example format. (Pay attention to the punctuation in the example.)

Even if only a single key is triggered, you must add a small delay like 0.1 seconds because there may be conflicts with rapidly consecutive rewards.

Click the Add Mapping button to save the mapping.

You can clear all mappings using the Clear Mappings button.

When everything is ready, you can start the server by clicking the Start button.

If you want to temporarily stop the server without closing the program, use the Stop button.

Enjoy using the program!

Note: To ensure the browser part of the program is working, after clicking the bookmark, open the console by pressing Ctrl+Shift+J on the page. If you see the message "Chat container found. Messages are being monitored...", it means the program is running smoothly.
